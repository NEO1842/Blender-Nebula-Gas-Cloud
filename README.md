# Blender-Nebula-Gas-Cloud

A procedural nebula gas cloud scene created in Blender using Shader Nodes.

# Overview

This project is a Blender scene designed to create a nebula gas cloud visual.

It uses Shader Nodes to generate mysterious gas-like formations and vibrant, colorful cloud effects.

The scene can be freely modified and used for learning Blender shader workflows or for creating similar visual styles.

# Software Requirement

Blender Version 3.6 LTS

# Rendering Settings

Engine: Cycles

Sampling: 1024

Denoising: Enabled

Original Inspiration: Ducky 3D  

Tutorial: https://www.youtube.com/watch?v=rASz8cpm-54&t=203s
